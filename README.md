## Cooltizbe - schedule application for IITU on iOS platform

## Getting Started

1. Clone the repo

```bash
$ git clone https://github.com/aitemr/cooltizbe.git
$ cd cooltizbe
```

2. Install dependencies from [Cocoapods](https://cocoapods.org/)

```bash
$ pod install
```
